# Branch Design — discharge enforcement

## Rites

- `restart-rite-46-writ-enforcement-counter`: converts the provisional writ into an enforceable docket and grants the enforcement clerk/seal.
- `restart-rite-47-review-list-erasure`: crosses Seventh Sunrise off the death-review list after a condition-driven review event.
- `restart-rite-48-exit-identity-check`: makes the exit admit the claimant as living by public identity or Linxi's B-07/0 shelter.
- `restart-rite-49-living-exit-turnstile`: binds the door to public minutes or the corpse-appeal copy after a door-static event.
- `restart-rite-50-seventh-sun-exit-ledger`: records the actual discharge and opens outside pursuit/escort pressure.

## Events

- `restart-event-46-review-window-reopens`: condition-triggered after enforcement counter; asks whether to protect Seventh Sunrise from review by public minutes or shelter her under Linxi's B-07/0 slot.
- `restart-event-48-door-minutes-static`: condition-triggered after exit identity check; attacks the door's copy of the hearing record.

## Costs / risks

Rites use existing proof cards so the current hand can fill them, but fictionally require public minutes, supply-chain rebuttal, pulse receipts, stamps, and the provisional writ. Events charge influence/favor branches and set remembered flags.

## State memory

- public enforcement vs B-07/0 shelter
- review name erased publicly vs hidden under Linxi
- door bound by public minutes vs appeal copy / supply contradiction
- exit pass held openly by Seventh Sunrise vs carried through Linxi shelter
- next route: escorted discharged witness vs pursuit of a temporary living cargo

## Ignored branch consequence

If pressure events are not answered, subsequent rites stay unavailable through flags (`restart_review_window_answered`, `restart_door_minutes_static_answered`).

## Post-turn-20 rule

Events use `trigger: {"condition": true}` and explicit flags with `NOT` guards; no pure turn trigger is used.
