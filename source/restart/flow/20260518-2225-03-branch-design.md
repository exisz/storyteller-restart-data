# 03 Branch Design — playable choices and state

Rite 10: Tunnel Transfer Register
- Choice A saves Linxi's memory route (`restart_saved_linxi_memory_route`).
- Choice B preserves Yue's access route (`restart_preserved_yue_access_route`).
- Common state: `restart_tunnel_transfer_registered`, `card-restart-eighth-wristband`, `card-restart-memory-ampoule`.

Event 11: Bed Number Broadcast
- Choice A has Linxi memorize B-07/0 (`restart_linxi_memorized_bed_zero`).
- Choice B has Yue stamp it (`restart_yue_stamped_bed_zero`).
- Common opener: `restart_bed_number_broadcast_answered`.

Rite 11: Bed Zero Signature Room
- Choice A forces Yue to admit the zero-bed transfer (`restart_yue_admitted_bed_zero`).
- Choice B hides the bed number in the radio (`restart_bed_number_hidden_in_radio`).
- Common state: `card-restart-real-bed-number`, `card-restart-blank-death-certificate`.

Event 12 opens the next branch via `restart_signature_pressure_decided`, preserving whether the player promised rescue or demanded Yue's debt truth.

Rite 12 is included so the next Player is not left at a cliff edge after resolving event 12.
